Vue.js 事件处理器

事件监听可以使用v-on指令:

通常情况下,需要使用一个方法来调用JavaScript方法;
v-on 可以接收一个定义的方法来调用;

可以用内联JavaScript语句;

事件修饰符

Vue.js为v-on提供了事件修饰符来处理DOM事件细节;
如:event.preventDefault()组织默认行为,或event.stopPropagation() 阻止冒泡;

  • .stop
  • .prevent
  • .capture
  • .self
  • .once

按键修饰符

Vue 允许为v-on 在监听键盘事件时添加按键修好符;

Vue为最常用的按键提供了别名;

全部按键的别名

  • .enter
  • .tab
  • .delete (捕获”删除键”和”退格键”)
  • .esc
  • .space
  • .up
  • .down
  • .left
  • .right
  • .ctrl
  • .alt
  • .shift
  • .meta
